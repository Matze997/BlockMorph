# BlockMorph

A PocketMine/Altay Plugin to morph yourself into a block

# Command

To morph yourself into a block use this command:
``/blockmorph [BlockID|BlockDamage]`` or ``/blockmorph hand``

To remove your morph use this command:
``/blockmorph remove``

# Permission
``cmd.blockmorph``
